# Not Enough Tooltips (don't let the repo name fool you!)

A lightweight **Fabric** mod for Minecraft 1.21+ that displays a sleek, animated popup showing item statistics whenever you switch items in your main hand.

## Features
- **Dynamic Stats**: Displays Attack Damage and Durability.
- **Enchantment List**: Shows all current enchantments on the item.
- **Fade Animation**: Smooth fade-out effect so the UI doesn't clutter your screen.
- **Lightweight**: Built with performance in mind using native Minecraft rendering.

## Installation
1. Make sure you have **Fabric Loader** installed for 1.21.1.
2. Download the latest `.jar` from the [Releases](https://github.com/Marmarek-here/Item-Tooltip-MC-Mod/releases) section.
3. Drop the file into your `.minecraft/mods` folder.
4. Requires **Fabric API**.
